Jak to działa?

Ta zakładka to żywa integracja Spring PetClinic z GitHub Copilot SDK (Python).

  1. Twoje pytanie trafia do Spring (:8080).
  2. Spring proxy przekazuje je do Python FastAPI (:8081).
  3. Mikroserwis uruchamia Copilot SDK z custom tools.
  4. AI wywołuje narzędzie get_available_vets, które odpytuje żywy endpoint /vets tej aplikacji.
  5. AI analizuje dane i odpowiada po polsku.
Demo: Wymagany uruchomiony serwis Python.
