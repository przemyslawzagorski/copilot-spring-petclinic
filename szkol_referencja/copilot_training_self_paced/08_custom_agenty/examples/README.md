# Examples — gotowe pliki `.github/agents`

Ten folder zawiera gotowy pack do ćwiczeń z orchestracji agentów:

- `.github/agents/feature-builder.agent.md`
- `.github/agents/researcher.agent.md`
- `.github/agents/implementer.agent.md`

## Jak użyć

1. Skopiuj pliki do głównego `.github/agents/` w repo, na którym ćwiczysz.
2. Otwórz Copilot Chat i wybierz agenta `Feature Builder`.
3. Uruchom zadanie typu: "research -> implement".

## Uwaga

Nazwy w `agents:` w `feature-builder.agent.md` muszą być zgodne 1:1 z polami `name` w subagentach.
